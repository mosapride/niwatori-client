# 庭箱

[箱庭](hako.v-kurore.com)のclientサイドプロジェクト。

clientサイトなのでOSSでもいいと公開。

※apiサーバーコードは非公開。

Youtuberのコミュニティーを目的としたサイト。

## development

```bash
npm install
npm start
```

## release

```bash
npm install
npm run build -- --aot=true --prod=true --sourceMap=false --outputPath=/usr/share/nginx/html
```
